<h1>osTicket - Configuring Roles & Permissions</h1>
In this tutorial, you will observe how to create a role and grant permissions.<br />


<h2>Environments and Technologies Used</h2>

- Microsoft Azure (Virtual Machines/Compute)
- Remote Desktop
- osTicket Platform

<h2>Operating Systems Used </h2>

- Windows </b> (Windows 11 PRO)


<h2>Steps</h2>

- Step 1: Log in to osTicket as Administrator. Go to Admin Panel -> Agents -> Roles -> Add Role
- Step 2: Add Permissions 
- Step 3: Add Tasks
- Step 4: Add knowledgebase and create role
- Step 5: Observe how the role (Supreme Admin) is created with every permission
